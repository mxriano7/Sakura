Um desejo não se limita à vontade de realizá-lo, afinal, o sonho das pessoas não tem fim.
Fim que não se encontra enquanto andarmos por este mundo com um propósito; fim que apenas chega quando tudo o que há para viver realmente acaba — tornando as lembranças nossos maiores tesouros.

Grandes coisas podem ser conquistadas enquanto estamos aqui, mas a única que eu realmente desejaria obter é você.
Cada passo rumo ao sucesso seria apenas por você. Afinal, tendo o seu coração, eu jamais temeria esse fim silencioso, pois meus sonhos estariam apenas começando.

E sobre aquela história de que o fim chega só quando tudo o que temos para viver se acaba... Pois bem, eu menti.
O que temos para viver, jamais acabará.
